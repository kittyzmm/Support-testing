## Описание проекта

Проект представляет собой реализацию модуля для работы с комплексными числами на языке Python. Модуль включает класс `ComplexNumber` для представления комплексных чисел и выполнения арифметических операций, а также вспомогательные функции для анализа и вывода данных.

## Функциональные возможности

Модуль поддерживает:
- создание комплексных чисел в форме *a + bi*;
- арифметические операции: сложение (`+`), вычитание (`−`), умножение (`×`);
- вычисление модуля (величины) комплексного числа;
- поиск числа с наибольшим модулем в списке;
- форматированный вывод комплексного числа с учётом знака мнимой части.

## Структура кода

- **Класс `ComplexNumber`** — основной класс для работы с комплексными числами:
  - конструктор `__init__(real, imaginary)`;
  - методы арифметических операций: `__add__`, `__sub__`, `__mul__`;
  - метод `magnitude()` для вычисления модуля;
  - метод `__str__()` для строкового представления.
- **Функция `find_largest_magnitude(numbers_list)`** — находит число с максимальным модулем в списке.
- **Функция `print_summary(real_part, imag_part)`** — выводит действительную и мнимую части числа в читаемом формате.

## Пример использования
```python
num1 = ComplexNumber(3, 4)
num2 = ComplexNumber(1, -2)
print("Сложение:", num1 + num2)  # Вывод: 4 + 2i
print("Модуль:", num1.magnitude())  # Вывод: 5.0
```
